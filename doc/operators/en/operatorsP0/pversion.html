pversion


Prints the current Pandore version number.


Synopsis

pversion

Description

pversion prints the current Pandore version number as a string, for example:

PANDORE 6.0.0 (2006-04-13)

Result

No result value.

Examples

  • Print the current Pandore version: 
       pversion
