pfftdeconvolution


Performs deconvolution of image by kernel.


Synopsis

pfftdeconvolution [im_in1|-] [im_in2|-] [im_out|-]

Description

pfftdeconvolution performs a deconvolution the input image im_in1 with the kernel given in the input image im_in2. The size of the kernel image im_in2 must be lower or equal than the input image im_in1.

The deconvolution uses the frequency domain. It consists in a complex division of the Fourier transform of the two input images:

  • imi=fft(im_in) / fft(im_in2)
  • im_out=iift(imi);

where / is the complex division.

Inputs

  • im_in1: a 2D image.
  • im_in2: a 2D image (same type as im_in1 and a size lower or equal to im_in1).

Outputs

  • im_out: a float image of the same size as the input image im_in1.

Result

Returns SUCCESS or FAILURE.
