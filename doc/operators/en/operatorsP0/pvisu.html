PANDORE Version 6 GREYC-IMAGE

pvisu


Displays Pandore, png, bmp or jpeg files.


Synopsis

pvisu [options] [-m mask] [im_in|-]

Description

pvisu is a graphical image display for jpeg, png, bmp and Pandore files. Any Pandore file are accepted:

  • images;
  • regions map;
  • graphs,
  • collections.

All tools are accessible through menu or by shortcut key.

To draw lines onto the input image, use pdraw operator.

Inputs

  • im_in: an image, a region map or a graph.

Parameters

  • -nofork is used to block pvisu until exit. 
       pvisu -nofork tangram.pan
  • Any Qt options (see Qt documentation). For example: 
       pvisu -style -nofork motif tangram.pan

Result

Returns the process ID (PID) of the related process or FAILURE.

Examples

  • Visualizes the image tangram.pan: 
       pvisu tangram.pan
  • Visualizes the contents of the center part of the tangram.pan image (size 50x50): 
       pshapedesign 256 256 0 1 50 50 a.pan
   pim2rg a.pan m.pan
   pmask examples/tangram.pan m.pan | pvisu
